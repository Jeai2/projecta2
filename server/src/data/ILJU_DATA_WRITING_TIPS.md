# 일주론 데이터 작성 팁

## 구조 변경 완료 ✅
- `personality` + `tendency` → `characteristic` (특성)으로 통합
- UI: "특성" 하나의 카드로 표시 (`IljuResult`)
- API/서비스: `characteristic` 있으면 사용, 없으면 `personality`+`tendency` 합쳐서 반환

## 현재 상태
- **갑자일주**: 긴 버전 예시 (characteristic만 사용)
- **나머지**: `characteristic` + `tendency`(legacy) 혼재 → 서비스에서 자동 통합
- **추가 작업**: 각 일주별로 `tendency` 제거 후 `characteristic`을 더 길게 확장 권장

## 작성 팁 (20~30대 여성 타겟)

### 1. 특성 (characteristic)
**목표 길이**: 150~250자

**작성 포인트**:
- 성격과 성향을 자연스럽게 연결하여 작성
- 구체적인 예시나 상황을 포함 (예: "주변 사람들에게...", "때로는...")
- 단점이나 주의사항도 함께 언급 (예: "하지만 때로는...", "다만...")
- 감성적이고 공감할 수 있는 표현 사용

**예시 (갑자)**:
```
"뛰어난 리더십과 추진력을 갖춘 성격으로, 목표를 향해 끈기 있게 나아가는 편입니다. 
적극적이고 진취적인 성향을 지니고 있어 새로운 도전을 즐기며, 변화를 두려워하지 않습니다. 
일단 결정하면 끝까지 포기하지 않는 강한 의지력을 갖추고 있어, 주변 사람들에게 신뢰받는 
리더 역할을 자연스럽게 맡게 됩니다. 하지만 때로는 너무 앞서나가려는 경향이 있어, 
주변 상황을 고려하지 못할 수 있으니 팀워크와 소통을 중시하는 것이 중요합니다."
```

### 2. 진로 (career)
**목표 길이**: 150~250자

**작성 포인트**:
- 구체적인 직업명 5~7개 제시
- 왜 그 분야에 적합한지 설명
- 어떤 능력이 발휘되는지 언급
- 성공 포인트나 주의사항 포함

### 3. 배우자 (spouse)
**목표 길이**: 150~250자

**작성 포인트**:
- 이상적인 관계 형태 설명
- 가정에서의 모습
- 함께 하는 것의 의미
- 조화를 위한 조언 포함

### 4. 재물운 (wealth)
**목표 길이**: 120~200자

**작성 포인트**:
- 재물 축적 방법과 스타일
- 투자 성향
- 주의해야 할 점
- 좋은 시기나 방법 제시

### 5. 건강 (health)
**목표 길이**: 120~200자

**작성 포인트**:
- 오행 불균형에 따른 주의할 부위
- 구체적인 건강 관리 방법
- 생활습관 조언
- 계절별 또는 상황별 주의사항

## 언어 스타일
- **공감형 표현**: "~하는 편입니다", "~한 경향이 있습니다"
- **구체적 묘사**: "때로는", "특히", "다만" 등으로 세밀한 설명
- **긍정적이되 현실적**: 장점과 함께 개선점도 제시
- **20~30대 여성 친화적**: 친근하고 이해하기 쉬운 언어

## 주의사항
- 각 필드를 독립적으로 작성 (다른 필드 참조하지 말 것)
- 일주별로 고유한 내용 (중복 최소화)
- 최대한 길고 상세하게, 하지만 핵심만 담기
