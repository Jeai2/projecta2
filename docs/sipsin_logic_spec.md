# 십신 관계 계산 로직 스펙 (v0.2)

> **v0.2 업데이트 (2026-04-02)**: 기획자 피드백 반영. 천간합 처리, 지지 우선순위 의미, 반합 조건, 정량 측정 범위, 지장간 보류 등 핵심 기준 명확화.

## 목적
사주 종합 해석 페이지(SipsinV2)에서 십신을 단순 나열하는 것이 아니라,
천간/지지의 관계를 계산하여 에너지 흐름을 도출하는 로직을 백엔드에 구현한다.

---

## 기본 원칙

### 길신 / 흉신 분류 (정격 기준)
- 길신: 식신, 정재, 편재, 정관, 정인
- 흉신: 상관, 편관(칠살), 편인(효신), 겁재(양인)
- 1원칙: 길신은 보호, 흉신은 제거
- 외격/특수격은 별도 판단 (길흉신 역할이 바뀔 수 있음)

### 천간과 지지의 성질 차이
| 구분 | 천간 | 지지 |
|------|------|------|
| 의미 | 보여지고 드러나는 상황 | 실제 현실적 상황 |
| 합 | 합거(合去) 있음, 합화(合化) 없음 | 합거 없음, 합화(合化) 있음 |
| 기운 | 순수한 단일 기운 | 천간의 조합 (지장간) |
| 사건 | 합거만 발생 | 형충파해합 **모두** 발생 |

## 1. 천간 계산 로직

### Step 1. 천간합거 처리
- 천간합 조합: 甲己, 乙庚, 丙辛, 丁壬, 戊癸
- **합이 성립하면 두 천간은 "당장 쓸 수 없는 상태"로 처리 (소멸이 아님)**
  - 합거된 천간은 현재 흐름에서 **동결** — 작용하지 않는 것으로 처리
  - 추후 관법(특수처리)에서 되살아날 수 있음 → 1차 뼈대에서는 **꼬임 방지를 위해 배제**
- 일간(日干)이 합 대상일 경우에도 동일하게 동결 처리 (1차 기준)
  - TODO: 관법 기준 추가 시 일간 합거 특수 로직 구현

### Step 2. 남은 천간 흐름 추적
- 합거(동결) 후 남은 천간들의 십신을 순서대로 추출
- 상생 관계로 연결되는 십신 흐름을 체인으로 표현

### Output 구조 (천간)
```
{
  합거목록: [{ 간1: "戊", 십신1: "정관", 간2: "癸", 십신2: "비견(일간)", 상태: "동결" }],
  잔존천간: [{ 간: "乙", 십신: "상관" }],
  천간흐름: ["상관(을)"]
}
```

## 2. 지지 계산 로직

### ★ 핵심 원칙: 형충파해합은 모두 동시에 발생한다
- 우선순위 **형 > 충 > 파 > 해 > 합** 은 **해석의 우선순위**일 뿐
- 선순위가 성립해도 후순위가 무효화되지 않음 — **모든 관계가 동시에 일어남**
- 해석 시 형·충부터 먼저 읽고, 합은 나중에 읽는다는 의미

### Step 1. 형충파해합 관계 판별

#### 2-1. 충(冲) 처리
- 충 조합: 子午, 丑未, 寅申, 卯酉, 辰戌, 巳亥
- 충이 발생하면 두 지지 간의 흐름이 차단됨
- **소멸이 아닌 약화** — 종합사주에서는 "약화 ≒ 쓸 수 없다"로 간주 (정량 측정 X)
- 충 받은 쪽의 십신은 무력화 처리

#### 2-2. 합화(合化) 처리
- 육합: 子丑(土), 寅亥(木), 卯戌(火), 辰酉(金), 巳申(水), 午未(土/火)
- 삼합: 寅午戌(火), 亥卯未(木), 巳酉丑(金), 申子辰(水)
- 방합: 寅卯辰(木), 巳午未(火), 申酉戌(金), 亥子丑(水)
- **반합 성립 조건: 왕지(旺支)가 반드시 포함되어야 함**
  - 寅午戌 → 午가 왕지. 寅午(O), 午戌(O), 寅戌(X)
  - 亥卯未 → 卯가 왕지. 亥卯(O), 卯未(O), 亥未(X)
  - 巳酉丑 → 酉가 왕지. 巳酉(O), 酉丑(O), 巳丑(X)
  - 申子辰 → 子가 왕지. 申子(O), 子辰(O), 申辰(X)
- 합화 결과는 해당 오행의 십신으로 변환
- 합화로 인해 흉신이 길신으로 변하거나 길신이 흉신으로 변할 수 있음

#### 2-3. 형(刑) 처리
- 삼형: 寅巳申, 丑戌未
- 자형: 子子, 午午, 酉酉, 亥亥
- 형 발생 시 해당 십신 손상 처리

#### 2-4. 파(破)·해(害) — 1차 보류
- 우선순위 판정 구조에 자리만 확보
- 세부 조합은 TODO (A 강화 단계)

### Step 2. 지지 생흐름 추적
- 충으로 무력화된 구간을 제외하고 상생 흐름 체인 추출
- 흉신이 길신을 생하는 구조 → 흉신의 와해/제거 효과로 해석
- 길신이 길신을 생하는 구조 → 에너지 강화로 해석
- **충으로 차단되어도 흐름 자체는 남아 있음** — 단, 충을 건 쪽에 의해 변덕/불안정이 생길 수 있음

### Output 구조 (지지)
```
{
  충목록: [{ 지1: "申", 십신1: "편인", 지2: "寅", 십신2: "식신", 결과: "무력화" }],
  합화목록: [{ 지1: "亥", 지2: "寅", 합화오행: "木", 결과십신: "식신" }],
  형목록: [],
  지지흐름: ["편인(신) → 비견(해) → 식신(인)"],
  무력화: ["식신(인) — 편인(신)의 충으로 무력화"],
  해석참고: "비견→식신 흐름은 남아있으나, 편인에 의해 변덕이 생길 수 있음"
}
```

## 참고 예시

### 예시 1: 1987년 3월 15일 오시 (丁卯년 癸卯월 癸亥일 戊午시)
- 일간: 癸水
- 천간: 戊癸합거 → **정관(戊)** + 월간(癸) 동결
  - 잔존 흐름: 비견(癸, 일간) → 편재(丁, 년간)
- 지지: 겁재(亥) → 식신(卯) → 편재(午)
  - 亥卯 육합 木局 → 겁재가 식신으로 뻗어나감
  - 겁재(흉신)가 식신(길신)을 생 → 흉신 와해 효과

### 예시 2: 1995년 12월 7일 인시 (乙亥년 丁亥월 壬申일 壬寅시)
- 일간: 壬水
- 천간: 丁壬합거 → 정재(丁) + 일간(壬) 동결
  - 잔존 흐름: 상관(乙)만 남음
- 지지: 편인(申) → 비견(亥) → 식신(寅)
  - 申寅 충 발생 → 식신(寅) 무력화
  - 亥寅 육합 木局 → 충과 합이 **동시에 발생** (해석은 충 우선)
  - 실제 해석: 편인(申) → 비견(亥) → 식신(寅) 흐름은 남아있으나, 편인에 의해 식신이 무력화되어 비견의 식신 방향에 변덕이 생길 수 있음
