## 기본 구조

1. **삼전은 사과 중에서 판단한다.** — 사과(四課)를 기반으로 삼전을 구한다.
2. **삼전은 사과를 10과로 판단한다.** — 사과 4개를 10과 중 하나로 분류해 판단한다.

---

## 극(剋)과 적(賊)

3. 재료에는 **극**과 **적**이 사용됨.
4. 극과 적은 둘 다 **오행 극**을 말하지만, **방향**이 다름.

| 구분       | 의미                        | 예시             |
| ---------- | --------------------------- | ---------------- |
| **극(剋)** | 천반이 지반을 극함 (天克地) | 천반 木, 지반 土 |
| **적(賊)** | 지반이 천반을 극함 (地克天) | 천반 土, 지반 木 |

---

## 삼전 계산 흐름

5. 사과의 **4부분(1과~4과)**에서 **극**과 **적**이 있는 부분을 확인한다.
6. **극**과 **적**의 개수 로직을 보고 판단하여 삼전의 **초전**에 사과 중 하나를 초전으로 만든다.
7. **극**과 **적**을 통해서 **10과**로 나눈다.

---

## 10과 종류 및 매칭 규칙

8. **10과의 종류**
   - 원수과, 중심과, 지일과, 섭해과, 요극과, 묘성과, 별책과, 팔전과, 복음과, 반음과

9. 사과를 10과(원수과~반음과)로 판단한다. **과는 중복되지 않고 오로지 1개의 과만 매칭된다.**

---

## 불비

16. **1과 4과가 같거나**, **2과 3과가 같은** 경우를 불비라 한다.

---

## 중전·말전 매칭

17. 기본적으로 삼전의 **중전**과 **말전**은 다음과 같이 매칭한다.

- **중전**: 초전의 천반을 천지반도 **지반**에서 찾아 → 해당 행의 지반, 천반, 둔간, 천장을 중전으로
- **말전**: 중전의 천반을 천지반도 **지반**에서 찾아 → 해당 행의 지반, 천반, 둔간, 천장을 말전으로

> ex. 초전 천반이 戌 → 천지반도에서 지반=戌인 행 찾아 → 그 행의 천반, 천장, 둔간을 중전으로

## 10과 상세

### 원수과

- **조건**: 1과 2과 3과 4과 중 하나가 **극**을 하는 상황일 때
- **초전**: 해당 **극**이 되는 과를 초전으로 매칭

### 중심과

- **조건**: 1과 2과 3과 4과 중 하나가 **적**을 하는 상황일 때
- **초전**: 해당 **적**이 되는 과를 초전으로 매칭

### 지일과

- **조건**: 1과 2과 3과 4과 중에 **적**이 두 개가 있는 경우
- **초전**:
  - 일진 천간이 **양천간**(甲, 丙, 戊, 庚, 壬) → 두 개 중 천반이 **양**인 과
  - 일진 천간이 **음천간**(乙, 丁, 己, 辛, 癸) → 두 개 중 천반이 **음**인 과

### 지일과2

- **조건**: 1과 2과 3과 4과 중에 **극**이 두 개 이상일 경우
- **초전**: 지일과와 동일
  - 일진 천간 양 → 천반이 **양**인 과
  - 일진 천간 음 → 천반이 **음**인 과

### 섭해과

- **조건**: 극이 두 개 이상 **OR** 적이 두 개 이상일 때, **일진 천간 음양에 맞는 과가 없을 때**
  - 천간 양인데 극/적 과가 모두 천반 음
  - 천간 음인데 극/적 과가 모두 천반 양
- **초전** (우선순위):
  1. 극/적 과 중 지반이 **寅申巳亥(인신사해)**인 과
  2. 없으면 지반 **子午卯酉(자오묘유)**
  3. 둘 다 없으면 지반 **辰戌丑未(진술축미)**

### 요극과

- **조건**: 극과 적이 1·2·3·4과 안에 없을 경우 (사과 내부에 극/적 없음)
- **재료**: 1과 일간 오행 ↔ 2·3·4과 천반(상신) 오행의 극 관계
  - **탄사(彈射)** = 적: 1과 일간이 2·3·4과 천반을 극함
  - **호시(蒿矢)** = 극: 1과 일간이 2·3·4과 천반에 극당함
- **초전**:
  - 호시·탄사가 있는 2·3·4과 중 선택
  - **호시 vs 탄사 동시 존재 시**: 호시 우선 (탄사 2개, 호시 1개여도 호시 초전)
  - **탄사 또는 호시 두 개 이상** 시: 지일과와 동일 — 일진 천간 음양에 맞는 천반 음양 과
    - 예: 甲(양) 일진, 2과 양 탄사, 4과 음 탄사 → 2과 초전

### 묘성과

- **조건**: 원수과, 중심과, 지일과, 지일과2, 섭해과, 요극과, 불비도 아닐 경우
- 일진 천간 음양에 따라:

**ⓐ 일진 천간 양**(甲, 戊, 丙, 庚, 壬)

- 천지반도에서 **酉** 지반 찾아 → 그 행의 천반, 둔간, 천장을 **초전**으로
- **중전** = 사과 3과, **말전** = 사과 1과

**ⓑ 일진 천간 음**(乙, 丁, 己, 辛, 癸)

- 천지반도에서 酉 지반 찾고 → 그 酉에 속한 **천반**을 다시 천지반도 지반에서 찾음 → 그 행의 지반, 천반, 둔간, 천장을 **초전**으로
- **중전** = 사과 1과, **말전** = 사과 3과

### 별책과

- **조건**: 극·적이 사과에 없고, 요극과 아님, **불비**(1·4과 같음 또는 2·3과 같음)일 때

**ⓐ 일진 천간 양**

- 천간별 천지반도 지반: 甲·壬→未, 丙→戌, 戊→丑, 庚→辰
- 해당 지반 행의 지반, 천반, 둔간, 천장을 **초전**으로
- **중전 = 1과, 말전 = 1과** (고정)

**ⓑ 일진 천간 음**

- 일지가 속한 삼합 그룹에서 **삼합 순서 상 다음 글자**를 천지반도 지반에서 찾음
- 해당 행의 지반, 천반, 둔간, 천장을 **초전**으로
- **중전 = 1과, 말전 = 1과** (고정)
- ex. 일지 亥 → 해묘미 중 다음 卯 / 일지 卯 → 未 / 일지 未 → 亥

### 팔전과

- **조건**: 사과 중 지반·천반·둔간·천장이 같은 모습이 두 개인 경우 (ex. 1과=3과·2과=4과 또는 1과=4과·2과=3과)
- **중전 = 1과, 말전 = 1과** (고정)

**ⓐ 일진 천간 양**: 1과 상신(1과 천반) 별로 초전 매칭

| 1과 천반 | 천지반도 지반 |
| -------- | ------------- |
| 子       | 寅            |
| 丑       | 卯            |
| 寅       | 辰            |
| 卯       | 巳            |
| 辰       | 午            |
| 巳       | 未            |
| 午       | 申            |
| 未       | 酉            |
| 申       | 戌            |
| 酉       | 亥            |
| 戌       | 子            |
| 亥       | 丑            |

**ⓑ 일진 천간 음**: 4과 상신(4과 천반) 별로 초전 매칭

| 4과 천반 | 천지반도 지반 |
| -------- | ------------- |
| 子       | 戌            |
| 丑       | 亥            |
| 寅       | 子            |
| 卯       | 丑            |
| 辰       | 寅            |
| 巳       | 卯            |
| 午       | 辰            |
| 未       | 巳            |
| 申       | 午            |
| 酉       | 未            |
| 戌       | 申            |
| 亥       | 酉            |

### 복음과

- **조건**: 1과 일간의 기궁과 1과 상신(1과 천반)이 같을 경우

**ⓐ 초전**
- 우선순위 1: 적, 우선순위 2: 극
- 극·적 없으면: 양천간 → 1과, 음천간 → 3과

**ⓑ 중전**
- 초전 지반이 1과 지반(기궁)과 자형(辰午酉亥) → 3과
- 초전 지반이 3과 지반(일지)과 자형(辰午酉亥) → 1과
- 초전 지반 子 → 천지반도 卯 행
- 초전 지반 卯 → 천지반도 子 행
- 초전 지반 삼형(寅巳申丑戌未) → 삼형 다음: 寅→巳, 巳→申, 申→寅 / 丑→戌, 戌→未, 未→丑

**ⓒ 말전**
- 중전 지반이 자형(辰午酉亥) → 중전과 충인 지반 행 (ex. 辰→戌)
- 중전 지반 子 → 천지반도 午 행
- 중전 지반 卯 → 천지반도 酉 행
- 초전 자형 & 중전 육형(子卯): 중전 子→말전 卯행, 중전 卯→말전 子행

### 반음과

- **조건**: 점시(정단)의 지반이 월장된 천반과 충을 이룰 때

**ⓐ 초전**
- 극·적 있으면: 적 우선, 극 차순위로 1~4과 중 초전
- 극·적 없으면: 일지 삼합 그룹의 역마(생지)를 천지반도 지반에서 찾아 초전
  - 해묘미→巳, 인오술→申, 사유축→亥, 신자진→寅
  - ex. 일진 辛亥일, 극·적 없음 → 해묘미 그룹, 역마 巳 구간 초전

**ⓑ 중전**
- 초전이 적/극 매칭 → 초전 천반을 천지반도에서 찾아 중전
- 초전이 역마 매칭 → 3과 중전

**ⓒ 말전**
- 초전이 적/극 매칭 → 중전 천반을 천지반도에서 찾아 말전
- 초전이 역마 매칭 → 1과 말전
