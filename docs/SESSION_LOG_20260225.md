## 1. 오늘의 운세 — 천을귀인 × 길한 시점 교집합

**파일**: `src/components/results/TodayFortuneResult.tsx`

### 변경 내용

기존에 천을귀인 시점(사용자 일간 기준 2개 지지 시간대)을 그대로 보여주던 것을, iching 길한 포인트의 시간(`data.iching?.lucky?.time`)과 **겹치는 부분만** 표시하도록 수정.

### 추가된 헬퍼 함수 (컴포넌트 외부)

- `BRANCH_HOURS` — 12지지별 시간(hour) 매핑 (子:[23,0], 丑:[1,2], ...)
- `parseIchingTimeToHours(timeStr)` — "오전 5–9시", "밤 11시–오전 3시" 등 자연어 시간 문자열을 `{start, end}` 시(hour)로 파싱
  - 지원 포맷: 오전 X–Y시, 오후 X–Y시, 새벽 X–Y시, 오전 X시–오후 Y시, 밤 X시–오전 Y시, 밤 X시–새벽 Y시, 밤 X시–자정, 밤 X시–밤 Y시, 밤 X시 이후
- `buildHourSet(start, end)` — start~end 범위의 hour 집합 생성 (자정 경계 처리)

### 로직 흐름

1. `data.iching?.lucky?.time ?? fortune.lucky.time` → iching 길한 시간 문자열
2. `parseIchingTimeToHours` → `{start, end}`
3. `buildHourSet` → 시간 Set
4. `gwiinBranches`에서 Set과 겹치는 branch만 필터링 → `overlappingBranches`
5. 겹치는 branch만 SVG 다이얼 하이라이트 + 시간 범위 표시
6. 라벨: "천을귀인 × 길한 시점"

## 8. 년주비교 로직 분석

**파일**: `server/src/services/couple-ohaeng.service.ts` → `calcNyeonjuAnalysis()`

### 흐름

1. **Step a** (년간끼리 십신 상호비교)
   - `sipsinH[myYearGan][partnerYearGan]` + 역방향
   - 두 십신 조합 → `SIPSIN_COMPAT_MAP` → 긍정/부정/중립

2. **Step b** (일간→자기 년간 십신)
   - `sipsinH[myDayGan][myYearGan]` + 상대도 동일
   - 두 십신 조합 → `SIPSIN_COMPAT_MAP` → 긍정/부정/중립

3. **a + b 합산**
   - 긍정+긍정 → 강 / 부정+부정 → 약 / 그 외 → 중

4. **Step c** (년지 관계)
   - 방합 → 강, 삼합 → 강
   - 충/형/원진/귀문 → 약
   - 없음 → 중

5. **최종**: `ab + c` → "강강"~"약약" (9가지)

### 핵심 테이블

- `SIPSIN_COMPAT_MAP` (1062~1081줄): 55개 십신 쌍 → true(긍정)/false(부정) 매핑
- `FINAL_STRENGTH` (프론트): 9가지 최종 등급 → 뱃지 색상
- `NYEONJU_COMPAT_DESC` (프론트): 9가지 최종 등급 → 요약 문구
