2026-04-27 논의 기반 — 홈 화면 콘텐츠 진단 및 개선 방향.

**Hero H1 문제 진단**: "무엇이 궁금하세요?"는 CS 창구 언어로 판정. sajurog의 차별점인 "자기 이해 도구" 포지션과 정렬되지 않음. 사용자를 질문자 포지션에 놓으면 서비스가 답변 기계로 인식됨.

개선 방향 — 공감형 권장: "당신이 어떤 사람인지, 사주에서 찾아드려요." 탐색의 주체를 서비스로 가져와 MBTI/심리테스트 유저 감각과 연결.

**Hero 서브카피 문제 진단**: "사주·운세·진로·인연 — 무엇이든 물어보세요"는 기능 목록 나열. 경쟁사와 차별화 없음. 브랜드 세계관이 아닌 기능 카탈로그로 읽힘.

개선 방향 권장: "수천 년의 명리학이, AI를 통해 당신의 언어로 말합니다." — AI+명리학 조합을 전면에 내세우는 포지셔닝.

**마스코트 "야등이" 위험 신호**: 야등이라는 캐릭터명이 "전통 명리학의 깊이" 차별점과 충돌 가능. 기능 카드 한 줄에만 등장하면 맥락 없는 캐릭터가 됨. 마스코트의 브랜드 내 포지션 확정이 콘텐츠보다 우선순위. 유지한다면 보조적 역할로 한정 권장.

**기능 카드 공통 문제**: 기능 설명이 기능은 말하지만 경험을 말하지 않음. "일주론으로 보는 내 일주"는 동어반복. "다가오는 한 해의 흐름을 점검"은 중립적이나 무감각. 개선 방향: 독자의 내면 질문을 대신 말해주는 언어.

**AI 상담 진입 전략**: placeholder "오늘 무엇이 궁금하세요?" → 빈 텍스트박스 앞 사용자 정지 현상 발생. 예시 질문 칩/탭 노출 권장. 예시 질문은 검색 키워드가 아닌 혼자 있을 때 실제로 떠오르는 생각 형식으로 작성.

**Why:** 경쟁사(네이버/카카오/점신/포스텔러) 대비 차별점이 "운세 알림"이 아닌 "자기 이해 도구"이므로, 모든 콘텐츠 언어가 이 포지션을 강화해야 함.

**How to apply:** 향후 홈 화면 카피 작성 시 H1과 서브카피의 정렬 여부를 먼저 체크. 기능 카드는 기능 설명이 아닌 경험 예고 언어로 작성. 마스코트 등장 여부는 브랜드 포지션 확정 후 판단.
